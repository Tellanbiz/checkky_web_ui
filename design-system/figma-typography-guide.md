# Checkky Dashboard - Typography Guide for Figma

## Font Information

### Primary Font
- **Font Name**: Inter
- **Fallback**: System UI fonts (SF Pro on Mac, Segoe UI on Windows)
- **Source**: Google Fonts or system default

### Font Sizes Used

| Element | Size (px) | Size (rem) | Weight | Tailwind Class |
|---------|-----------|------------|---------|----------------|
| Large Heading | 30px | 1.875rem | Bold (700) | `text-3xl font-bold` |
| Section Heading | 24px | 1.5rem | Semibold (600) | `text-2xl font-semibold` |
| Card Title | 20px | 1.25rem | Semibold (600) | `text-xl font-semibold` |
| Subheading | 18px | 1.125rem | Medium (500) | `text-lg font-medium` |
| Body Text | 16px | 1rem | Regular (400) | `text-base` |
| Small Text | 14px | 0.875rem | Regular (400) | `text-sm` |
| Caption | 12px | 0.75rem | Regular (400) | `text-xs` |
| Button Text | 14px | 0.875rem | Medium (500) | `text-sm font-medium` |
| Navigation | 14px | 0.875rem | Medium (500) | `text-sm font-medium` |
| Large Metrics | 36px | 2.25rem | Bold (700) | `text-4xl font-bold` |
| Medium Metrics | 24px | 1.5rem | Bold (700) | `text-2xl font-bold` |

### Font Weights Used
- **Regular**: 400 (Normal text)
- **Medium**: 500 (Labels, buttons, navigation)
- **Semibold**: 600 (Headings, card titles)
- **Bold**: 700 (Large headings, metrics)

### Line Heights
- **Tight**: 1.25 (Headings)
- **Normal**: 1.5 (Body text)
- **Relaxed**: 1.625 (Long form content)

## Setting Up in Figma

### 1. Install Inter Font
- Download Inter from Google Fonts
- Install on your system
- Or use Figma's built-in Inter font

### 2. Create Text Styles
Create these text styles in Figma:

**Headings:**
- H1: Inter, 30px, Bold (700)
- H2: Inter, 24px, Semibold (600)
- H3: Inter, 20px, Semibold (600)
- H4: Inter, 18px, Medium (500)

**Body:**
- Body Large: Inter, 16px, Regular (400)
- Body Medium: Inter, 14px, Regular (400)
- Body Small: Inter, 12px, Regular (400)

**UI Elements:**
- Button: Inter, 14px, Medium (500)
- Label: Inter, 14px, Medium (500)
- Caption: Inter, 12px, Regular (400)

**Data Display:**
- Metric Large: Inter, 36px, Bold (700)
- Metric Medium: Inter, 24px, Bold (700)
- Metric Small: Inter, 20px, Semibold (600)

### 3. Color Combinations
- Primary text: #0F172A (dark)
- Secondary text: #64748B (muted)
- Success: #059669
- Warning: #D97706
- Error: #DC2626
